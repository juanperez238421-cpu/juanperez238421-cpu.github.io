## A/B Testing Analysis

This project evaluates an A/B experiment to compare the conversion rate between a control group and a treatment group.

The objective was to determine whether the new version produced a statistically significant improvement in conversion.

### Tools and Project Type

![Python](https://img.shields.io/badge/PYTHON-295F98?style=for-the-badge&logo=python&logoColor=white)
![Pandas](https://img.shields.io/badge/PANDAS-295F98?style=for-the-badge&logo=pandas&logoColor=white)
![Statistics](https://img.shields.io/badge/STATISTICS-295F98?style=for-the-badge)
![AB Testing](https://img.shields.io/badge/A_B_TESTING-295F98?style=for-the-badge)

### Key Questions

- Which variant had the highest conversion rate?
- Was the observed difference statistically significant?
- Should the new version be implemented?
- What business recommendation can be made from the experiment?

### Methodology

- Data loading and validation
- Conversion rate calculation
- Hypothesis definition
- Statistical test for proportions
- Business conclusion based on p-value and observed effect

### Main Insights

The treatment group showed a slightly higher conversion rate, but the statistical test did not provide enough evidence to confirm a significant improvement. The recommendation was not to implement the new version as a final change based only on this experiment.

## Funnel and Cohort Retention Analysis

This project analyzes user behavior through a conversion funnel and weekly cohort retention.

The objective was to identify where users drop off in the journey and evaluate how retention behaves over time.

### Tools and Project Type

![SQL](https://img.shields.io/badge/SQL-295F98?style=for-the-badge)
![Python](https://img.shields.io/badge/PYTHON-295F98?style=for-the-badge&logo=python&logoColor=white)
![Power BI](https://img.shields.io/badge/POWER_BI-295F98?style=for-the-badge&logo=powerbi&logoColor=white)
![Cohort Analysis](https://img.shields.io/badge/COHORT_ANALYSIS-295F98?style=for-the-badge)

### Key Questions

- How many users move from first visit to purchase?
- At which stage does the largest drop-off occur?
- How stable is retention across weekly cohorts?
- What business actions could improve conversion and retention?

### Methodology

- User journey definition
- Funnel construction by event stage
- Drop-off calculation
- Cohort grouping by registration month
- Weekly retention rate calculation
- Dashboard communication

### Main Insights

The funnel revealed relevant user loss before purchase, while cohort retention remained relatively stable across weeks. Improving friction points in intermediate stages could increase total conversion.

**Repository:** [View project](https://github.com/juanperez238421-cpu)
