Excel Executive Sales Dashboard — Walmart Sales Analysis

This Excel project analyzes Walmart sales data to identify performance drivers, store efficiency and departmental contribution. The workflow transforms raw transactional records into an executive dashboard and business report.

The analysis uses Power Query/Excel for cleaning and ETL, calculated fields for modeling, pivot tables over more than 400,000 rows and slicers for dynamic filtering by department and period.

Key Questions

  • Which departments contribute the most to revenue and operational performance?
  • Which areas show the highest sales efficiency per square meter?
  • What data-quality issues affect executive reporting?

Excel/Walmart dashboard animation using P2V3.mp4.

Findings and Recommendations

  • Efficiency: Pantry & Basics was identified as the efficiency leader with strong sales per square meter.
  • Revenue drivers: Fresh Food and Household Goods were key business contributors.
  • Data quality: Undefined sales categories indicate an opportunity to improve classification at the source.

ConnectaTel Customer Analytics with Python

This Python project analyzes customer behavior for ConnectaTel using plan, user and usage datasets. The workflow integrates demographic data from 4,000 users with 40,000 usage records to identify consumption patterns, outliers and customer segments.

The analysis covers EDA, missing-city treatment, sentinel correction, date handling, statistical consumption profiles, outlier detection and segmentation by age and usage level.

Key Questions

  • What customer behaviors define high-risk or high-value users?
  • Which usage patterns indicate outliers or power users?
  • How can customers be segmented for retention and upselling?

E-commerce SQL Funnel and Cohort Retention

This SQL project analyzes a MercadoLibre-style e-commerce journey from initial visit to final transaction during January–August 2025. The core focus is multi-step funnel conversion and cohort retention analysis.

The analysis identifies an approximately 85% drop between product selection and checkout, evaluates retention after registration and compares cohort stability over time.

Key Questions

  • Where do users drop off in the e-commerce funnel?
  • How does retention behave after registration?
  • Which cohorts are more stable over time?

Landing Experiment A/B Test

This experimentation project compares landing page versions A and B using conversion rate, average spend among converted users and reproducible statistical testing.

The dataset contains 40,000 users. Version B achieved a higher conversion rate, moving from 12.57% to 15.96%, and higher average spend among converted users.

Key Questions

  • Does landing version B improve conversion?
  • Does version B increase average spend among converted users?
  • Do traffic source or user type influence conversion?
