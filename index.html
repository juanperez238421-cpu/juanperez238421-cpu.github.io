Commercial Performance Dashboard

This project analyzes the commercial performance of a retail business using key metrics such as revenue, net profit, profit margin, number of orders, average ticket, product category, customer segment, and country.

The objective was to build an executive dashboard in Power BI to help stakeholders identify business performance, profitability opportunities, and commercial trends.

Tools and Project Type

Power BI DAX Business Intelligence Dashboard

Methodology

  • Data validation and preparation
  • Data modeling in Power BI
  • KPI creation using DAX
  • Dashboard design with filters, cards, and business visuals
  • Executive summary with business recommendations

Main insight: The business showed solid overall performance, but profitability varied by product and category. Some products generated strong revenue and profit, while others required a review of pricing, costs, or commercial strategy.

A/B Testing Analysis

This project evaluates an A/B experiment to compare the conversion rate between a control group and a treatment group.

The objective was to determine whether the new version produced a statistically significant improvement in conversion.

Tools and Project Type

Python Pandas Statistics A/B Testing

Methodology

  • Data loading and validation
  • Conversion rate calculation
  • Hypothesis definition
  • Statistical test for proportions
  • Business conclusion based on p-value and observed effect

Main insight: The treatment group showed a slightly higher conversion rate, but the statistical test did not provide enough evidence to confirm a significant improvement.

Funnel and Cohort Retention Analysis

This project analyzes user behavior through a conversion funnel and weekly cohort retention.

The objective was to identify where users drop off in the journey and evaluate how retention behaves over time.

Tools and Project Type

SQL Python Power BI Cohort Analysis

Methodology

  • User journey definition
  • Funnel construction by event stage
  • Drop-off calculation
  • Cohort grouping by registration month
  • Weekly retention rate calculation

Main insight: The funnel revealed relevant user loss before purchase, while cohort retention remained relatively stable across weeks.
