E-commerce SQL Funnel and Cohort Retention

This SQL project analyzes a MercadoLibre-style e-commerce journey from initial visit to final transaction during January–August 2025. The core focus is not generic SQL: it is multi-step funnel conversion and cohort retention analysis.

The analysis identifies an approximately 85% drop between product selection and checkout, evaluates retention after registration, and compares cohort stability over time. The SQL implementation uses CTEs, complex joins, cohort logic, conditional aggregation and null-safe calculations.

Landing Experiment A/B Test

This experimentation project compares landing page versions A and B using conversion rate, average spend among converted users and reproducible statistical testing.

The dataset contains 40,000 users, with 19,982 users in version A and 20,018 in version B. Version B achieved a higher conversion rate, moving from 12.57% to 15.96%, and higher average spend among converted users, from 61.09 to 68.75.

The analysis includes a Welch t-test for spend, a z-test for proportions for conversion, and chi-square tests for traffic source and user type. The executive recommendation is to implement version B while monitoring segments and traffic-source behavior.
