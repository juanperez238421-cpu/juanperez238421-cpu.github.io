Dashboard Ejecutivo de Ventas en Excel — Walmart Sales Analysis

Este proyecto en Excel analiza datos de ventas de Walmart para identificar impulsores de desempeño, eficiencia por tienda/departamento y contribución al negocio. El flujo transforma registros transaccionales en un dashboard ejecutivo y reporte de negocio.

El análisis usa Power Query/Excel para limpieza y ETL, campos calculados para modelado, tablas dinámicas sobre más de 400,000 filas y segmentadores para filtrar dinámicamente por departamento y periodo.

Preguntas clave

  • ¿Qué departamentos aportan más al revenue y al desempeño operativo?
  • ¿Qué áreas muestran mayor eficiencia de ventas por metro cuadrado?
  • ¿Qué problemas de calidad de datos afectan el reporting ejecutivo?

Animación del dashboard Excel/Walmart usando P2V3.mp4.

Hallazgos y recomendaciones

  • Eficiencia: Pantry & Basics fue identificado como líder de eficiencia con alto desempeño por metro cuadrado.
  • Impulsores de revenue: Fresh Food y Household Goods fueron contribuyentes clave del negocio.
  • Calidad de datos: Las categorías Undefined muestran una oportunidad de mejora en la clasificación de origen.

Analítica de Clientes ConnectaTel con Python

Este proyecto en Python analiza el comportamiento de clientes de ConnectaTel usando datasets de planes, usuarios y uso. El flujo integra datos demográficos de 4,000 usuarios con 40,000 registros de uso para identificar patrones de consumo, outliers y segmentos de clientes.

El análisis incluye EDA, tratamiento de ciudades faltantes, corrección de sentinels, manejo de fechas, perfiles estadísticos de consumo, detección de outliers y segmentación por edad y nivel de uso.

Preguntas clave

  • ¿Qué comportamientos definen usuarios de alto riesgo o alto valor?
  • ¿Qué patrones de uso indican outliers o power users?
  • ¿Cómo segmentar clientes para estrategias de retención y upselling?

Funnel y Retención por Cohortes en E-commerce con SQL

Este proyecto SQL analiza un recorrido e-commerce tipo MercadoLibre desde la visita inicial hasta la transacción final durante enero-agosto de 2025. El foco central es conversión por etapas del funnel y retención por cohortes.

El análisis identifica una caída aproximada del 85% entre selección de producto y checkout, evalúa retención después del registro y compara estabilidad de cohortes en el tiempo.

Preguntas clave

  • ¿En qué etapa del funnel se pierden más usuarios?
  • ¿Cómo se comporta la retención después del registro?
  • ¿Qué cohortes son más estables en el tiempo?

Experimento A/B para Landing Page

Este proyecto de experimentación compara las versiones A y B de una landing page usando tasa de conversión, gasto promedio de usuarios convertidos y pruebas estadísticas reproducibles.

El dataset contiene 40,000 usuarios. La versión B logró una mayor tasa de conversión, pasando de 12.57% a 15.96%, y mayor gasto promedio entre usuarios convertidos.

Preguntas clave

  • ¿La versión B mejora la conversión?
  • ¿La versión B aumenta el gasto promedio de usuarios convertidos?
  • ¿La fuente de tráfico o el tipo de usuario influyen en la conversión?
